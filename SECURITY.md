# Security Policy

## Project Status: School Prototype
This repository contains a **school project and prototype**. It is intended for educational purposes and is not a production-ready application.

### Important Security Context
* **Local Use Only:** This software is designed to be run strictly in a local environment. It is not built to be exposed to the internet or hosted on a public server.
* **Zero Dependencies:** This project is built without external third-party libraries or dependencies, significantly reducing the risk of supply-chain vulnerabilities.
* **Maintenance:** This project is **not actively maintained**. We do not plan to release regular updates or patches. 

---

## Reporting a Vulnerability

While this is a prototype, i value security. If you find a **very serious issue** that could impact a user's local system, please report it privately.

### How to Report
Please do not open a public GitHub Issue for security vulnerabilities. Instead, please contact the maintainer:
* **Email:** github-security-issu.2y3i3@passmail.net 

### Response Policy
Because this is a student project, please keep the following in mind:
* **Prioritization:** Updates will only be considered for critical security flaws. 
* **Timeline:** There is no guaranteed response time, but i will attempt to acknowledge receipt of the report.
* **Fixes:** Minor issues or bugs that do not pose a significant security risk will likely not be addressed.
